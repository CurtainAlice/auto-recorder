# 智能表格复制工具

## 功能说明

这是一个Windows自动化工具，可以**识别网页表格的行号/序号**，自动复制指定范围的数据到Excel。

### 核心功能

1. **OCR识别序号** - 自动识别网页表格的行号
2. **自动定位** - 根据行号找到屏幕上的位置
3. **智能选择** - 自动选择指定范围的行
4. **批量复制** - 一次最多100条，自动粘贴到Excel
5. **记录数量** - 自动统计已复制的数据条数
6. **保存Excel** - 数据自动保存到Excel文件

### 适用场景

- DataWorks电子表格数据导出
- 网页表格批量复制到Excel
- 需要按行号范围选择数据的场景

## 使用步骤

### 运行程序

1. 双击运行 `智能表格复制工具.exe`

### 手动复制模式（推荐）

```
1. 登录DataWorks，打开电子表格
2. 运行程序，选择"手动复制模式"
3. 在DataWorks中选择要复制的行（最多100行）
4. Ctrl+C 复制
5. 在程序窗口按 Enter 粘贴到Excel
6. 重复操作直到完成
7. 输入 quit 退出
```

### 自动复制模式（需要OCR）

```
1. 确保已安装Tesseract OCR引擎
2. 运行程序，选择"自动复制模式"
3. 输入起始行号（如：82）
4. 输入结束行号（如：181）
5. 程序自动：
   - 截取屏幕识别行号
   - 定位到起始行
   - 按住Shift选择到结束行
   - Ctrl+C复制
   - 粘贴到Excel
6. 记录已复制数量，继续下一批
```

## 技术说明

- **OCR引擎**: Tesseract OCR（支持中英文识别）
- **屏幕截图**: Pillow库
- **模拟操作**: pyautogui库
- **Excel生成**: openpyxl库

## 注意事项

1. 自动模式需要安装Tesseract OCR引擎
2. 每次最多复制100条数据
3. Excel文件自动保存在output目录
4. 程序支持中文显示
