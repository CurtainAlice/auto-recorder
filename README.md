# 鼠标键盘录制回放自动化工具

## 功能说明

这是一个Windows自动化工具，可以录制你的鼠标键盘操作，然后回放这些操作，实现自动化。

### 主要功能

1. **录制操作** - 记录你的鼠标点击、键盘输入等操作
2. **回放操作** - 按照录制的操作自动执行
3. **循环执行** - 可以设置循环次数，重复执行操作
4. **保存/加载** - 保存录制的操作序列，下次可以加载使用
5. **GUI界面** - 可视化操作，易于使用

### 适用场景

- 从网页表格批量复制数据到Excel
- 自动化重复性的鼠标键盘操作
- 批量处理相同的操作流程

## 使用步骤

### Windows打包步骤

1. 安装Python 3.8+
   - 下载：https://www.python.org/downloads/
   - 安装时勾选"Add Python to PATH"

2. 双击运行 `build.bat`

3. 等待打包完成，在 `dist` 文件夹找到 `鼠标键盘录制回放工具.exe`

### 运行程序

1. 双击运行 `鼠标键盘录制回放工具.exe`

### 操作流程

#### 1. 录制操作

1. 打开你要操作的程序（如DataWorks网页）
2. 在工具中点击"开始录制"或按 F9
3. 执行你的操作：
   - 点击表格单元格
   - 选择多行数据
   - Ctrl+C 复制
   - 切换到Excel
   - Ctrl+V 粘贴
4. 完成后点击"停止录制"或按 F9
5. 程序自动保存录制

#### 2. 回放操作

1. 设置循环次数（如：10次 = 复制1000条数据）
2. 点击"开始回放"或按 F10
3. 程序自动执行录制的操作
4. 按 F12 可随时停止

#### 3. 保存/加载录制

- 点击"保存录制"可以保存操作序列文件
- 点击"加载录制"可以加载之前保存的操作

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| F9 | 开始/停止录制 |
| F10 | 开始回放 |
| F12 | 退出程序 |

## 注意事项

1. 录制时注意操作顺序，回放会按照完全相同的顺序执行
2. 如果网页加载速度慢，可以在录制时适当增加等待时间
3. 回放时确保目标程序在相同位置
4. 按 F12 可以随时停止回放
