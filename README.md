# 智能表格复制工具

## 功能说明

这是一个Windows工具，可以从网页表格复制数据到Excel，每次最多100条。

### 核心功能

1. **手动复制** - 你选择和复制数据，程序自动粘贴到Excel
2. **自动保存** - 数据自动保存到Excel文件
3. **记录数量** - 自动统计已复制的数据条数
4. **批量处理** - 支持多次复制，累计数据

### 适用场景

- DataWorks电子表格数据导出
- 网页表格批量复制到Excel
- 需要按行号范围选择数据的场景

## 使用步骤

### 运行程序

```
1. 双击运行 "智能表格复制工具.exe"
```

### 操作流程

```
1. 登录DataWorks，打开电子表格
2. 运行程序，选择"手动复制模式"
3. 在DataWorks中选择要复制的行（最多100行）
4. Ctrl+C 复制
5. 在程序窗口按 Enter 粘贴到Excel
6. 重复操作直到完成
7. 输入 quit 退出
```
